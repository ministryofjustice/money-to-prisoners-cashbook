{% trans 'How Nov 2nd policy changes will affect you' %}

{% trans 'No more bank transfers' %}

{% trans 'A policy change from November 2nd means that senders have to ‘Send money to someone in prison’ using a debit card.' %}

{% trans 'They will no longer be able to send money by bank transfer or send in cash, cheques or postal orders by post.' %}

{% trans 'If they don’t have a bank account, we’re encouraging them to set one up. If they can’t use the service, they can apply to the governor for an ‘exemption’.' %}

{% trans 'You may see a temporary rise in complaints because of these changes.' %}

{% trans 'What is the disbursements cap?' %}

{% trans 'A policy change from November 2nd means only £50 a week can be sent out to each person.' %}

{% trans 'Example: They can send up to £250 a week, if it’s to 5 different people.' %}

{% trans 'If they want to send out more' %}

{% blocktrans %} A resident must apply for approval from the governor if they need to send out more than £50 to a person in a week. In exceptional circumstances, this might be granted. It must be applied for each time by filling in a form which is attached to their disbursement before handing in. {% endblocktrans %}

{% trans 'What you need to do' %}

{% trans 'If the resident’s application is approved, make the disbursement. If it is not approved, send the application form back with a reason why.' %}

{% trans 'Exceptional circumstances could be:' %}

  • {% trans 'paying off debts if you have no external bank account' %}
  • {% trans 'sending financial support to family' %}
  • {% trans 'paying for a legitimate service, e.g. legal fees, dentistry' %}
  • {% trans 'operating a business, if unconvicted' %}
  • {% trans 'transferring your money from prison to a new or existing bank account' %}
