{% trans 'How to track digital disbursements in SOP' %}

{% trans 'Disbursements will normally appear in SOP ‘accounts payable’ and the ‘general ledger (GL)’ a couple of days after they have been processed via the digital cashbook.' %}

{% trans 'In the ‘accounts payable’ enquiry screen, you can view all details of the individual disbursements (see screenshots below).' %}

{% trans 'The transactions are listed in the GL report but are always dated for the last day of the month, not the date they were uploaded to SOP.' %} {% trans 'As every transaction uploaded within a month will have the same GL date, it’s recommended that the GL report is run daily for the NOMIS to SOP reconciliation.' %}

{% trans 'Alternatively, it is possible to run a ‘discoverer report’ to see the details of a disbursement.' %} {% trans 'The date parameters of the discoverer report must be for the whole month to ensure the disbursement transactions are included.' %}
