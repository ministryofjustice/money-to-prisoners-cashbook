{% trans 'When a prisoner wants to send money out of prison (for example, to relatives or charities), use this new disbursement service to digitally process the payment request.' %} {% trans 'You can only use this service to process bank transfers and cheques.' %} {% trans 'It works alongside the existing service of sending money out by post.' %} {% trans 'This service is free – there are no postage or processing fees.' %}

{% trans 'To start using this service, go to ‘manage prisoner money’, use your digital cashbook login and click on ‘digital disbursements’.' %}

{% trans 'How it works' %}

1. {% trans 'Prisoner fills in the paper form and has their identity confirmed' %}

{% url 'disbursements:paper-forms' as link %} {% blocktrans trimmed %} The wing officer gives the correct disbursement form to the prisoner to fill in. {% endblocktrans %} {% trans 'Prisoner fills in the form and the wing officer signs it to confirm the prisoner’s identity.' %} {% trans 'It can be put into an envelope to make it secure.' %}

{% trans 'Using the digital disbursement service in prisons with kiosks works slightly differently to those without.' %}

{% trans 'How to set up the new ‘digital disbursement survey’ on kiosks' %}

{% trans 'Follow normal security procedures' %}

{% trans 'When a prisoner submits a disbursement, please follow your prison’s normal security procedures.' %}

{% trans 'Prisoner identity must be verified in person and hub staff must confirm the authorised signature on the paper form, just as before.' %} {% trans 'If forms need to be checked by security staff, before or after going to the business hub, this process should stay the same.' %}

{% trans 'Paper forms' %}

{% url 'disbursements:paper-forms' as link %} {% blocktrans trimmed %} All paper forms have been updated. {% endblocktrans %}

{% trans 'Print out and use whichever of the four forms suit your prison procedures' %}:

  • {% trans 'digital disbursements (bank transfer and cheques)' %}
  • {% trans 'postal orders and cash' %}
  • {% trans 'postal orders only' %}
  • {% trans 'cash only' %}

2. {% trans 'Hub staff #1 enters paper form details into the new disbursement tool' %}

{% url 'disbursements:start' as link %} {% blocktrans trimmed %} A business hub staff member enters the details from the paper form into the Enter a payment request section of this tool. {% endblocktrans %}

{% trans 'This tool is connected to NOMIS so it shows a prisoner’s up-to-date account information.' %} {% trans 'Money will come out of the prisoner’s private cash account.' %} {% trans 'But, as before, money can be manually moved in NOMIS by hub staff, to a private account from spends or savings, then sent out.' %}

3. {% trans 'Hub staff #2 double-checks and confirms the details on the paper form and the details in the digital tool are the same' %}

{% trans 'When the payment request has been entered into the disbursement tool, the paper form must be handed over to a colleague who has access to the tool.' %}

{% trans 'They need to log in to the tool to double-check the details entered into the tool match the paper form details.' %} ({% trans 'Login details for the digital cashbook and the disbursement tool are the same.' %} {% trans 'Ask your business hub manager if you have access problems.' %})

{% url 'disbursements:pending_list' as link %} {% blocktrans trimmed %} Then, if the details match, click on Confirm payment and sign the paper form. {% endblocktrans %}

{% trans 'The confirmed request automatically goes to SSCL for processing.' %}

{% trans 'Who can’t confirm payment' %}

{% trans 'A payment can’t be confirmed by the person who' %}:

  • {% trans '‘entered’ the payment request' %} – {% trans 'the process needs a different member of staff to check the details' %}
  • {% trans 'is last to make a ‘change’ or ‘edit’ to the request' %}

4. {% trans 'Payment request is automatically deducted from NOMIS then goes to SSCL which sends out payment and remittance advice' %}

{% trans 'SSCL sends the payment to the recipient by bank transfer or cheque, as requested by the prisoner.' %} {% blocktrans trimmed with letter=_('remittance advice')|wrapwithtag:'strong' %} It also sends out {{ letter }} which is a note sent to the recipient by post or email (both free). {% endblocktrans %} {% trans 'It names the prisoner who sent the money and how much they sent.' %} {% trans 'There’s a ‘recipient email’ box on the form for the prisoner to fill in.' %}

{% trans 'Sample remittance advice' %}

{% trans 'In 5-7 working days, either the payment is put directly into the recipient’s account by bank transfer or a cheque is received.' %} {% trans 'There’s no postage or processing charge to the prison, the prisoner or the recipient.' %}

{% trans 'Further information' %}

{% trans 'How to find things on SOP' %}

{% trans 'No more cheques to send out' %}

{% trans 'The business hub no longer needs to send out cheques.' %} {% trans 'When you’ve digitally processed a cheque payment using the disbursement tool, SSCL will write and post out the cheques.' %}

{% trans 'Notify the prisoner' %}

{% trans 'Once the payment request is in progress, the prisoner money team will email a notification to the prison staff so they can notify the prisoner.' %} {% url 'disbursements:submit_ticket' as link %} {% blocktrans trimmed %} Contact us if you don’t already get these notifications. {% endblocktrans %}

{% trans 'Use cash and postal orders' %}

{% trans 'You don’t have to use cash or postal orders if you don’t want to or don’t already offer these methods.' %} {% trans 'If you do offer them, the process is the same as before.' %} {% trans 'In the disbursement tool, you can only digitally process bank transfers and cheque payments.' %}

{% trans 'Track payment progress' %}

{% trans 'To track payment progress you’ll need the invoice number from the digital disbursement tool.' %} {% trans 'You’ll find this in the ‘payments made’ section.' %}

{% url 'disbursements:track-invoice' as link %} {% blocktrans trimmed %} Once you have the invoice number, enter it into SOP (into ‘accounts payable’) to see when the bank transfer has been made or the cheque sent out. {% endblocktrans %}

{% trans 'If the prisoner gets the recipient’s bank details wrong' %}

{% trans 'If the sort code is wrong, SSCL will cancel the bank transfer and email the prison in about 2 days.' %}

{% trans 'If the bank transfer gets rejected by the bank, then SSCL cancels it and emails the prison.' %} {% trans 'This takes about 10 days.' %}

{% trans 'How to cancel a cheque or see if it’s been cashed' %}

{% blocktrans trimmed with group_name='‘SOP P2P Accounts Payable’' type_name='‘MoJ payments - non staff’' %} To do both of these, you need to raise a Service Request (SR) to the group {{ group_name }} with type {{ type_name }}. {% endblocktrans %}

{% trans 'To cancel a cheque, tell SSCL immediately then raise the SR within 24 hours of the disbursement being sent.' %} {% trans 'SSCL has 5 working days to look into this, but they can’t guarantee a cancellation.' %}

{% trans 'Once the bank is notified, it takes up to 2 days for them to cancel a cheque up to which time the cheque can still be cashed.' %} {% trans 'You need to credit the prisoner manually in NOMIS to return their money.' %}

{% trans 'To see if a cheque has been cashed, raise the SR then ask SSCL if they can check with the bank.' %}

{% trans 'Make security staff aware' %}

{% trans 'Money sent out using this digital service is shown alongside money coming into the Prisoner Money Intelligence tool so please tell security staff.' %}

{% trans 'However, the intelligence tool won’t show postal orders, cash or cheques using the previous postal method.' %}
