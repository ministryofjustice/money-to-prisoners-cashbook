{% trans 'How to set up the new ‘digital disbursement survey’ on kiosks' %}

{% trans 'In prisons with kiosks, the digital disbursement process requires a kiosk ‘disbursement survey’ (the kiosk equivalent of the paper form already in use) to be filled in by a prisoner.' %}

{% trans 'This survey needs to be added to the kiosk before you can use the digital disbursement tool.' %}

{% trans 'Once the prisoner has filled in their payment request online at a kiosk, it will be sent to the business hub for you to process using the new digital service.' %}

{% trans 'How to upload and alter the ‘survey’ on the kiosk' %}

  1. {% trans 'Click on the ‘Surveys’ module, then the ‘Surveys’ button to open up current surveys' %}
  2. {% trans 'Click on the option ‘Active’ and untick the box as you change the settings' %} – {% trans 'remember to re-activate it once you’ve finished' %} – {% trans '‘Only allow assigned nominals to complete this survey’ must be ticked for this system to work' %}
  3. {% trans 'Click the option ‘Import’ at the bottom of the screen' %} – {% trans 'ignore the error message and click ‘OK’' %}
  4. {% trans 'Select the ‘Survey’ you want to upload from your USB device which should show in the ‘Surveys’ list' %}
  5. {% trans 'Click ‘Change’ to change parts of each section' %}
  6. {% trans 'Change the ‘Header’ section to fit your requirements' %} – {% trans 'for example, HMP Berwyn decided to only allow 2 cash transfers of a maximum of £20 each.' %} {% trans 'Your site may be different so you’ll need to change this.' %} {% trans 'If you do, you’ll also need to change later parts of the survey' %}
  7. {% trans 'To allow daily use of the survey, have the ‘Open-ended conditions’ set to re-activate every 1 day' %}
  8. {% trans 'The next section with the ‘Question set’ can be changed accordingly too' %}
  9. {% trans 'Remember to go back to the ‘Surveys’ button and untick the option ‘Active’ once you’ve changed the settings.' %} {% trans 'You must tick ‘Only allow assigned nominals to complete this survey’ for this system to work' %}
  10. {% trans 'Perhaps do a trial run to see if it all works?' %}
